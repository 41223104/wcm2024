5.主遊戲迴圈

def main():
    current_player = 'X'
    while True:
        display_board(board)
        print(f"Player {current_player}'s turn")
        row = int(input("Enter row (0, 1, 2): "))
        col = int(input("Enter column (0, 1, 2): "))

        # 檢查是否合法
        if 0 <= row < ROWS and 0 <= col < COLS and board[row][col] == ' ':
            board[row][col] = current_player
            if check_win(board, current_player):
                display_board(board)
                print(f"Player {current_player} wins!")
                break
            elif all(cell != ' ' for row in board for cell in row):
                display_board(board)
                print("It's a tie!")
                break
            else:
                # 切換玩家
                current_player = 'O' if current_player == 'X' else 'X'
        else:
            print("Invalid move! Try again.")

if __name__ == "__main__":
    main()

  • current_player = 'X'：初始化當前玩家為 'X'。

  • while True:：這是一個無限迴圈，直到有玩家贏得比賽或者平局才會結束。

  • display_board(board)：在每一輪遊戲開始時顯示當前棋盤的狀態。

  • print(f"Player {current_player}'s turn")：打印當前輪到哪個玩家。

  • row = int(input("Enter row (0, 1, 2): ")) 和 col = int(input("Enter column (0, 1, 2): "))：讓玩家輸入他們想要下子的位置。

  • 檢查是否合法：確保玩家的輸入是有效的，即檢查玩家是否在有效範圍內輸入並且該位置是否為空。

  • 如果輸入合法，則在棋盤上放置當前玩家的標記。

  • 使用check_win函式檢查遊戲是否結束，如果有一方贏得了比賽，則顯示相應的訊息並結束遊戲。

  • 如果沒有玩家贏得比賽，則檢查是否所有的格子都已經被填滿，如果是，則宣布平局並結束遊戲。

  • 如果遊戲還沒有結束，則切換到另一個玩家，然後繼續下一輪遊戲。

  • 如果玩家的輸入不合法，則提示玩家重新輸入。

運行這段程式碼時，它將模擬一個井字遊戲，並允許兩個玩家交替下棋，直到其中一方贏得比賽或者達到平局為止
